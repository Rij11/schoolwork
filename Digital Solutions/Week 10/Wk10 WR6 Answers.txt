Q1.
A Selector is the variable before the {} in CSS. It selects the element, tag, area that needs to be formatted. (EG: p, .center, p.center)
Declaration is a string that includes both a property and a property value. (EG: color: red;)
A Property is a set inside of the {}'s. The data before the : is the property while the data before the ; is the property value. (EG: color: red; (color is property, red is the property value))
A Value is the varaible that changes the design of the property. (Described above)
