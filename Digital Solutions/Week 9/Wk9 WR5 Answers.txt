Q1.
<!DOCTYPE html> is used set the language of the document to HTML.
<html> </html> is used to set the area where all of the html script will be written.
<head> </head> is a container used to store metadata and is placed within the html tag.
<title> </title> is placed within the head tag and is the text that appears on the browser tab.
<body> </body> is used to define the documents body. It contains headings, paragraphs, images, hyperlinks, etc.
